# Weather App

A simple weather application built using HTML, CSS, and JavaScript.  
It allows users to search for real-time weather information by entering any city name.

## Features

- Search weather by city name
- Supports global city search
- Displays city and country
- Shows temperature in Celsius
- Shows weather condition
- Shows humidity
- Shows wind speed
- Handles invalid city input
- Uses real-time weather data from API

## Tech Stack

- HTML
- CSS
- JavaScript
- OpenWeather API
- Git & GitHub

## Concepts Used

- DOM Manipulation
- Event Handling
- Fetch API
- Async/Await
- API Integration
- Error Handling

## How It Works

1. User enters a city name.
2. JavaScript sends a request to the OpenWeather API using `fetch()`.
3. The API returns real-time weather data.
4. The app displays temperature, condition, humidity, wind speed, city, and country.
5. If the city is invalid, an error message is shown.
